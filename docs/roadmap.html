Version {{ major_int }}.{{ minor_int }}.0

This version will focus on reducing the size of saved reports.

  • Use PHP's base_convert() to shorten hash length.
  • Use single character key names in the directories.txt file.
  • Allow for reports to be generated without the list of files and/or specific totals.
  • Do not save files for empty directories.
  • Version in settings.txt will be increased to 1.1.

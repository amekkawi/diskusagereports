Below is a tentative roadmap of future changes to Disk Usage Reports. There are no specific dates attached to the releases. To get a better understanding of how version numbers are incremented, see the About Version Numbers section of the documentation.

Version 1.1.0

  • Include sub directory count in totals.

Version 1.2.0

  • Improve CSS for better printing support.

Version 1.3.0

  • Use PHP's base_convert() to shorten hash length.
  • Use single character key names in the directories.txt file.
  • Version in settings.txt will be increased to 1.1.

Version 1.4.0

  • Use http://code.google.com/p/flot/ to display charts for tables.

Version 1.5.0

  • Allow reports to be compared against each other to show differences. For example, it will show if a directory increased/decreased in size.

Version 1.6.0

  • Optionally show bandwidth used by files by processing a custom web server log. The log would need to include the file system path of the file servered and the bytes sent.
  • Top 100 files by bandwidth used.

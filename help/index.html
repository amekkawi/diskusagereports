Troubleshooting

I am getting Error: An unknown error occurred (status: 0)

This can occur when a cross-domain HTTP security error is encountered. This can happen when your server has server-side redirects that effect the host name of the URL. For example, you may have a redirect that changes myhost.com to www.myhost.com. These can be difficult to diagnose, so please contact me at help@diskusagereports.com
